# NL to CLI Agent (nl-to-cli-agent)

ממיר הוראות בשפה טבעית לפקודות CLI ב‑Windows (גרסה V3 — Safe Mode).

**מטרה**
- לקבל הוראות בעברית או אנגלית ולהחזיר פקודת טרמינל מתאימה ב‑Windows.
- להגן מפני החזרה של פקודות מסוכנות (מחיקה, כיבוי, הפסקת תהליכים בכוח וכו').

**תכונות עיקריות**
- קריאה ל‑API של OpenAI דרך proxy (`AIWALL_API_KEY`).
- פרומפט מערכת מחמיר (`SYSTEM_PROMPT_V3`) לחסימת פקודות מסוכנות.
- ממשק משתמש ב‑Gradio עם דוגמאות בעברית.
- טיפול בסיסי ב‑`.env` והצגת הודעות שגיאה ידידותיות בעברית.

---

## דרישות מקדימות
- Python 3.10+
- חבילות: `openai`, `gradio`
- גישה ל‑API דרך `AIWALL_API_KEY` (proxy) — המפתח מאוחסן בקובץ `.env` בתיקייה של הפרוייקט.

---

## התקנה מהירה
בתיקיית הפרוייקט (`nl-to-cli-agent`) הרץ ב‑PowerShell:

```powershell
python -m venv .venv
& .\.venv\Scripts\Activate.ps1
pip install --upgrade pip
pip install openai gradio
```

הערה: אם יש `requirements.txt` או `pyproject.toml` הפרויקט יכול להשתמש בהם במקום התקנה ידנית.

---

## קובץ `.env`
צרו קובץ בשם `.env` בתיקיית הפרויקט עם השורה הבאה:

```
AIWALL_API_KEY=your_api_key_here
```

- `AIWALL_API_KEY` הוא המפתח שמשמש את הלקוח (`openai.OpenAI`) וכולל `base_url` מוגדר ל־`https://proxy.aiwall.org`.
- אל תדחוף את קובץ ה־`.env` לריפו ציבורי.

---

## הרצה
לאחר שהפעלת הסביבה והתקנת התלויות והוספת ה‑`.env` הרץ:

```powershell
python main.py
```

ברירת מחדל Gradio יפעיל שרת מקומי ב־`http://127.0.0.1:7861`.

---

## שימוש ודוגמאות
הכנס הוראה כמו:
- `מה כתובת ה-IP של המחשב שלי`
- `איזה תהליכים רצים`
- `הצג את גרסת Windows`

התוכנית מחזירה פקודת CLI (או הודעת שגיאה בעברית אם ההוראה ריקה או אסורה).

## אבטחה והתנהגות צפויה
- הפרוייקט מנסה לחסום החזרת פקודות מסוכנות דרך `SYSTEM_PROMPT_V3`. עם זאת, אין תחליף לבדיקת־קוד ובקרת שימוש — יש להיזהר לפני הרצת פקודות שמתקבלות מאינטליגנציה מלאכותית.
- אין להריץ פקודות מסוכנות שהתקבלו ישירות ללא ביקורת ידנית.
